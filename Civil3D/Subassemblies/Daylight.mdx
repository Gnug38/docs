## Overview

The **UXCAD Dynamic Curb** subassembly creates a configurable curb section with control over flange, flowline, top front, and top back geometry. It is designed to let the user build curb conditions using either fixed dimensions, grades, or target-driven control points.

This subassembly is useful for curb and gutter conditions where the curb shape needs to remain flexible while still maintaining consistent Civil 3D point, link, and shape coding.

## Primary Use Cases

- Standard curb and gutter sections
- Flange-to-flowline controlled curb layouts
- Flowline-to-top-front curb face control
- Top-front-to-top-back curb cap control
- Target-driven curb geometry
- Code-controlled corridor surfaces and quantities

## Attachment

The attachment point is located at the **Flange, FlowLine and Top Back Curb** of the curb section.

## Input Parameters

| Parameter | Display Name | Type | Default | Description |
| --- | --- | --- | --- | --- |
| Side | Side | Side | Right | Determines whether the daylight condition is placed on the left or right side. |
| DaylightType | Day Light Type | Daylight\_Type | Slope | Controls whether the daylight is driven by slope or offset distance. |
| DaylightSlopeType | Daylight Surface Type | Daylight\_Surface\_Type | Slope | Controls whether the daylight ties to the surface using slope or grade logic. |
| DaylightGrade | Daylight Grade | Grade | -2.00% | Grade used for daylight projection. |
| DaylightSlope | Daylight Slope | Slope | -2.00:1 | Slope ratio used for daylight projection. |
| DaylightDistance | Daylight Distance | Double | 10 | Offset distance used when daylighting by offset. |
| InsidePointCode | Inside Point Code | String | P1 | Code assigned to the inside daylight point. |
| OutsidePointCode | Outside Point Code | String | P2 | Code assigned to the outside daylight point. |
| LinkCode | Link Code | String | L1 | Code assigned to the daylight link. |

## Target Parameters

| Target Parameter | Type | Description | Required |
| --- | --- | --- | --- |
| DaylightSurface | Surface | Surface target used for daylight tie-in calculations. | Required |
| DaylightOffset | Offset | Optional offset target controlling daylight location. | Optional |
| DaylightElevation | Elevation | Optional elevation target controlling daylight elevation. | Optional |

## Behavior

The curb is built from the flange point and continues outward through the flowline, top front, and top back geometry. Each major curb segment can be controlled by a fixed height, fixed grade, or target parameter depending on the selected control method.

## Notes

Use this subassembly when the curb section needs to remain flexible across different design conditions without creating multiple separate curb subassemblies.
